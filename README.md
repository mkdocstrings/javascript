Javascript handler for mkdocstrings.

## Installation

With `pip`:
```bash
pip install mkdocstrings-javascript
```

With [`pipx`](https://github.com/pipxproject/pipx):
```bash
python3.7 -m pip install --user pipx
pipx install mkdocstrings-javascript
```
